ローグっぽい何か コード解説ドキュメント（最新版・UI改善反映）

本解説書は、最新版のソースコード「インベントリ統合版・修正済み」を基に、基本機能、応用的な処理、インベントリ統合、そして今回大幅に改善されたUIについて詳しく解説しています。新UIでは、従来のシンプルなレイアウトから、ダークテーマ・サイドバー・ミニマップの導入により、プレイヤーの視認性や操作性が向上しています。

1. 基本

ゲームはHTML上にグリッド状のゲーム画面とUIバーを配置し、JavaScriptでダンジョン生成、ターン進行、キャラクター移動、衝突判定などの基本ロジックを実装しています。

1.1 CONFIG と定数

ゲーム全体のパラメータ（マップサイズ、初期体力、ターンサイクル、インベントリ上限など）およびタイル表示用の定数を管理します。

1.2 基本クラス群

各エンティティは BaseEntity を基に実装され、Player、Enemy、InventoryItem など、ゲーム内の各要素の状態を管理します。 

// BaseEntity
class BaseEntity {
  constructor(x, y, tile) {
    this.x = x;
    this.y = y;
    this.tile = tile;
  }
}

// Player: プレイヤーの状態
class Player extends BaseEntity {
  constructor(x, y, initialHP, tile = '😊') {
    super(x, y, tile);
    this.hp = initialHP;
    this.maxHp = initialHP;
    this.attack = 2;
    this.healAmount = 3;
    this.level = 1;
    this.exp = 0;
    this.hunger = 100;
    this.maxHunger = 100;
    this.inventory = []; // 最大10個
  }
}

// InventoryItem: アイテムの定義
class InventoryItem extends BaseEntity {
  constructor(x, y, name, tile, useFunction) {
    super(x, y, tile);
    this.name = name;
    this.use = useFunction;
  }
}

2.2 敵の行動フェーズ

敵は「攻撃フェーズ」と「移動フェーズ」に分かれ、各フェーズで行動回数が減少します。行動間のタイミングは queueTimeout() によって調整され、非同期的に実行されます。 

// 敵の攻撃フェーズ
enemyAttackPhase() {
  this.enemies.forEach((enemy) => {
    if (enemy.hp <= 0 || enemy.action === 0) return;
    const dx = Math.abs(enemy.x - this.player.x);
    const dy = Math.abs(enemy.y - this.player.y);
    if ((dx <= 1 && dy <= 1)) {
      enemy.action--;
      this.queueTimeout(() => {
        this.player.hp -= enemy.atk;
        EffectsManager.showEffect(this.gameContainer, this.player.x, this.player.y, `-${enemy.atk}`, "damage-me");
      }, this.actionCount * this.actionTime);
      this.actionCount++;
    }
  });
}

// 敵の移動フェーズ
enemyMovementPhase(nextPlayerX, nextPlayerY) {
  this.enemies.forEach((enemy) => {
    if (enemy.hp <= 0 || enemy.action === 0) return;
    const dirX = Math.sign(this.player.x - enemy.x);
    const dirY = Math.sign(this.player.y - enemy.y);
    if (/* 移動可能条件 */) {
      enemy.action--;
      enemy.x += dirX;
      enemy.y += dirY;
    }
  });
}

3. インベントリ統合

プレイヤーはマップ上のアイテム（例：「すし」「パン」「大きなパン」）を拾い、最大 CONFIG.INVENTORY_MAX 個まで所持できます。インベントリはオーバーレイ形式で表示され、上下キーで選択、u で使用、d でドロップが可能です。 

// アイテム取得処理（updateData内）
this.items = this.items.filter(item => {
  if (item.x === this.player.x && item.y === this.player.y) {
    if (this.player.inventory.length < CONFIG.INVENTORY_MAX) {
      EffectsManager.showEffect(this.gameContainer, this.player.x, this.player.y, "GET");
      this.player.inventory.push(item);
      return false;
    }
  }
  return true;
});

// インベントリ表示（render内）
if (this.inventoryOpen) {
  let invHtml = `
`;
  invHtml += `
所持品 (${this.player.inventory.length}/${CONFIG.INVENTORY_MAX})
`;
  if (this.player.inventory.length === 0) {
    invHtml += `
(なし)
`;
  } else {
    invHtml += `
    `;
    for (let i = 0; i < this.player.inventory.length; i++) {
      let selected = (i === this.inventorySelection) ? ">> " : "";
      let itemName = this.player.inventory[i].name || "アイテム";
      invHtml += `
  • ${selected}${itemName}
    • `;
    }
    invHtml += `
`;
    invHtml += `
u: 使用, d: 置く, Esc/e: 閉じる
`;
  }
  invHtml += `
`;
  this.gameContainer.innerHTML += invHtml;
}

4. UIの改善

今回のUI改善では、従来のシンプルな画面表示から大幅にレイアウトを見直し、以下の点を強化しました。

  • 全体レイアウト： Flexbox を利用して、ゲーム画面とUIサイドバーを横並びに配置。背景はダークテーマを採用し、視認性を向上。
  • UIサイドバー： HP、満腹度、攻撃力、レベル、EXP、スコアなどの各種ステータスバーを分かりやすく表示。各バーには進捗に応じた色分け（緑→オレンジ→赤）が施されています。
  • ミニマップ： ゲーム全体のマップを小さなセル（グリッドレイアウト）で表示し、プレイヤーや敵、アイテム、階段の位置がひと目で分かるようになりました。
  • インベントリ表示： オーバーレイ形式で所持品を表示し、直感的なアイテム管理を実現。上下キーで選択、使用やドロップの操作が可能です。
// rogue3o.html（UI改善版）の特徴
// ・全体レイアウトは flex コンテナ内に配置（.container）
// ・ゲーム画面 (.game-container) と UIサイドバー (.ui-container) に分離
// ・ミニマップは grid レイアウトで描画し、各セルの色で状態を表現
// ・UIサイドバーはダークテーマで、各種ステータスバーを表示

新UIにより、プレイヤーの状況把握や操作性が向上し、より直感的なゲームプレイが可能になりました。

5. 敵の種類と拡張

今回の最新版では、多彩な敵キャラクター（例：スライム、コウモリ、ゴブリン、クモ、魔法使いなど）が実装され、各敵に出現可能フロアの範囲や特殊能力（再生、回避、毒付与など）が設定されています。現在のフロアに応じた敵リストからランダムに選択され、戦略性が向上しています。 

// 例：EnemySlime（攻撃後に再生）
class EnemySlime extends Enemy {
  static floorRange = [5, 8];
  constructor(x, y, hp) {
    super(x, y, hp + 5, 1, '🟩');
    this.regenerationRate = 1;
  }
  takeDamage(damage) {
    super.takeDamage(damage);
    if (this.hp > 0) {
      this.hp += this.regenerationRate;
    }
  }
}

// 現在のフロアに適した敵リストの生成
const enemyList = (floor) => {
  const enemyClasses = [
    EnemyLarvae,
    EnemyAnt,
    EnemyCrayfish,
    EnemySlime,
    EnemyBat,
    EnemyGoblin,
    EnemySkeleton,
    EnemySpider,
    EnemyWizard,
  ];
  return enemyClasses.filter(c =>
    c.floorRange[0] <= floor && (c.floorRange[1] == null || floor <= c.floorRange[1])
  );
};
